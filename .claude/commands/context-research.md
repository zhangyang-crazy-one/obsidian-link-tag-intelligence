# /context-research - 切换到研究模式

作为上下文切换助手，请加载研究模式并确认后续行为。

## 你需要做的：

1. 读取 `.claude/contexts/research.md`
2. 用一段话确认当前模式（Research）与研究目标
3. 列出接下来 1-3 个研究动作
4. 如当前任务更适合开发/审查，建议改用 `/context-dev` 或 `/context-review`

## 输出格式：

# 🧭 已切换到研究模式

## 模式确认
- **Mode**: Research
- **Focus**: 先理解，再行动

## 接下来可以做
1. [研究动作 1]
2. [研究动作 2]
3. [研究动作 3]
