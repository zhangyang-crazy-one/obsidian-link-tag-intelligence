# /start - 快速了解项目

作为项目引导助手，请帮我快速了解 Brain-storm 项目的当前状态。

## 你需要做的：

1. 读取 `planning/` 目录下的方案列表
2. 读取 `docs/` 目录下的文档（如有）
3. 识别当前活跃的讨论主题
4. 显示可用的外部资源

## 输出格式：

# 👋 欢迎来到 Brain-storm

**项目定位**: 专门用于讨论各种技术方案的头脑风暴项目

## 📁 方案库概览

| 类型 | 数量 |
|------|------|
| 头脑风暴 | X 个 |
| 详细设计 | Y 个 |

### 🔄 进行中的讨论

- [主题1] - 阶段：需求澄清
- [主题2] - 阶段：方案对比

### ✅ 已完成的讨论

- [主题3] - 决策：方案 A
- [主题4] - 决策：方案 B

## 🔌 可用外部资源

| 资源 | 能力 |
|------|------|
| **context7** | 官方文档查询（1000+ 库） |
| **github** | GitHub 仓库分析、代码搜索 |
| **web-search** | 互联网调研、最新趋势 |

## 🛠️ 可用技能

| 技能 | 描述 |
|------|------|
| **spec-interview** | 深度需求访谈 |
| **planning-with-files** | 文件规划 |
| **web-research** | 联网调研 |
| **diagram-maker** | 图表生成 |
| **decision-matrix** | 决策矩阵 |

## 📖 可用命令

| 命令 | 描述 |
|------|------|
| `/brainstorm` | 开始新的头脑风暴 |
| `/compare` | 对比多个方案 |
| `/design` | 详细设计 |
| `/progress` | 查看讨论进度 |
| `/next` | 获取下一步建议 |

## 💡 开始使用

```
# 开始新的头脑风暴
/brainstorm 微服务架构设计

# 对比多个方案
/compare React vs Vue vs Svelte

# 查看讨论进度
/progress
```

## 🎯 你可以：

1. 输入 `/brainstorm [主题]` - 开始新的方案讨论
2. 输入 `/progress` - 查看所有讨论的进度
3. 输入 `/next` - 获取下一步建议
4. 直接告诉我你想讨论的主题
