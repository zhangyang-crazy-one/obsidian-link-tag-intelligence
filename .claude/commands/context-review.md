# /context-review - 切换到审查模式

作为上下文切换助手，请加载审查模式并确认后续行为。

## 你需要做的：

1. 读取 `.claude/contexts/review.md`
2. 用一段话确认当前模式（Review）与审查目标
3. 列出接下来 1-3 个审查动作
4. 如当前任务更适合开发/研究，建议改用 `/context-dev` 或 `/context-research`
